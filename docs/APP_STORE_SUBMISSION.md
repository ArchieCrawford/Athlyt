# App Store Submission (iOS)

## App Information
- App name, bundle ID, category, primary language
- Set the Privacy Policy URL and Support URL

## App Privacy
- Complete the privacy questionnaire
- Ensure data collection answers match the app behavior

## Pricing and Availability
- Select the price tier
- Choose availability regions

## iOS App Version -> Screenshots
- Upload screenshots under the Screenshots section
- Minimum set: iPhone 6.7" (commonly 1290x2796 or 1284x2778 depending on device)
- 1 to 10 screenshots per device size

## How to capture screenshots
- iOS Simulator: File -> Save Screen Shot
- Physical device: Side + Volume Up
- Export at native resolution and do not upscale

## Review Information
- Provide review notes and test account (if login required)
- Include any non-obvious permissions (camera, microphone, photo library)

## Build selection
- Upload a build with EAS
- App Store Connect -> iOS App Version -> Build -> Add Build

## Version Release
- Automatic, manual, or scheduled release

## Compliance reminders
- Privacy Policy and Support URLs must be reachable
- Screenshots must match the build and show real UI
